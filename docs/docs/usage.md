# Usage

This guide covers the basics of using BoardGameTracker after installation.

## First Steps

After starting BoardGameTracker for the first time, you will be guided through the initial setup process.

## Managing Your Collection

### Adding Games

Games can be added by searching BoardGameGeek directly from the application. Navigate to **Games** in the sidebar and click the add button to search and import games.

### Logging Play Sessions

To log a play session:

1. Click **New Session** in the sidebar
2. Select the game you played
3. Add players and their scores
4. Save the session

### Managing Players

Navigate to **Players** to add and manage the people you play with. Each player has their own profile with play statistics and win rates.

## Features Overview

### Dashboard

The dashboard provides an overview of your gaming activity, including recent sessions, most played games, and player statistics.

### Shelf of Shame

Enable the Shelf of Shame feature in **Settings** to track games in your collection that haven't been played yet.

### Game Nights

Organize game nights with your group. Plan events and track which games were played during each session.

### Locations

Track the locations where you play your games to see where most of your gaming happens.

### Compare

Use the compare feature to compare statistics between different games or players side by side.
