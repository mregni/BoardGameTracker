## What is BoardGameTracker?

BoardGameTracker is a self-hosted web application for tracking your board game collection and play sessions.
Built with ASP.NET Core and React, it gives you full control over your data while providing a modern, responsive interface.

## Features

- **Game Collection** — Manage your board game library with details pulled from BoardGameGeek
- **Play Sessions** — Log plays with scores, players, and game-specific details
- **Player Management** — Track players, win rates, and play statistics
- **Dashboard** — Get insights into your gaming habits with charts and statistics
- **Shelf of Shame** — Keep track of unplayed games in your collection
- **Game Nights** — Organize and plan game nights with your group
- **Locations** — Track where you play your games
- **Multi-language** — Supports multiple languages via Crowdin

## Tech Stack

| Component | Technology |
|-----------|-----------|
| Backend | ASP.NET Core 8, Entity Framework Core |
| Frontend | React, TypeScript, TanStack Router & Query |
| Database | PostgreSQL |
| Deployment | Docker |
